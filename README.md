# Household Tasks Dashboard

A premium, interactive dashboard to track personal, medical, and work tasks.

## Features
- Dynamic task statistics and graphs
- Multi-dimensional filters (Category, Status, Priority, Search)
- Switchable Grid/List and Kanban Board views
- Theme support (Light/Dark mode)
- Google Sheet CSV parser with drag-and-drop support

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the dev server: `npm run dev`.
